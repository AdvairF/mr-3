## 2 BUGS DESCOBERTOS EM UAT (PAUSA #1-RETOMADA)

### BUG #1 (FK schema) — DESCOBERTO + FIXED em prod 2026-04-27

`eventos_andamento_contrato_id_fkey` apontava para `contratos(id)` (tabela inexistente — herança D-pre-8 de schema histórico) em vez de `contratos_dividas(id)`. Migration 004 com `ADD COLUMN IF NOT EXISTS` não recriou FK porque coluna `contrato_id` já existia de tentativa anterior. Sem este fix, INSERT em `eventos_andamento` falhava 100% silenciosamente — registrar evento na Fila estava 100% quebrado em prod desde Phase 7.13b Plan 01 ship. Fix aplicado manualmente em prod hoje via 4 blocos SQL. Migration 005 registra fix. Memory feedback `feedback_add_column_if_not_exists_skips_references.md` documenta lição.

### BUG #2 (saldo sistêmico) — DESCOBERTO + DOCUMENTADO (não fixed)

Motor de encargos lê das dívidas individuais (`dividas`) em vez de `contratos_dividas` (single source declarado). Drift confirmado em prod no contrato M L FRIOS (UUID 335a2ad2-9481-4836-a88a-55fbe6375827):
- `contratos_dividas`: multa 2%, honorários 10%
- `dividas` (individual): multa 30%, honorários 0%
- Motor exibe: saldo com multa 30% (errado).
