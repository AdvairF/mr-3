## Interaction Contracts

### ModuloContratos ‚Äî State Machine

```
view = "lista"         ‚Üí shows TabelaContratos + "Novo Contrato" button in header
view = "novo"          ‚Üí shows NovoContrato form (contrato header only)
view = "detalhe"       ‚Üí shows DetalheContrato (header + documentos list + "+ Adicionar Documento")
view = "parcela-detalhe" ‚Üí shows DetalheDivida for a parcela (navigated from inside DetalheContrato)
```

Navigation rules:
- "novo" back button ‚Üí "lista"
- "detalhe" back button ‚Üí "lista"
- Clicking a parcela row in DetalheContrato ‚Üí "parcela-detalhe" (sets `selectedParcela`)
- "parcela-detalhe" back button ‚Üí "detalhe" (returns to DetalheContrato, not "lista")
- `‚Üê Ver documento` in DetalheDivida ‚Üí re-opens DetalheContrato scrolled to the documento section (implementation: call `onVerContrato(contrato_id)` which sets view="detalhe")
- `‚Üê Ver contrato` in DetalheDivida ‚Üí same as above (both breadcrumbs land on DetalheContrato because DetalheDocumento is inline ‚Äî not a separate view)

### TabelaContratos ‚Äî Row Interactions

Primary visual anchor: `+ Novo Contrato` button in module header ‚Äî draws eye before table content.

- Click anywhere on row: opens DetalheContrato
- Hover: `background: "#f8fafc"` on row, `cursor: "pointer"`, transition `.15s`
- Pagination: 20 rows per page, same Anterior/Pr√≥ximo pattern as TabelaDividas
- Default sort: `created_at DESC` (most recent first)

Columns:

| # | Header | Width | Content |
|---|--------|-------|---------|
| 1 | Credor | auto | `credor.nome` or `‚Äî sem credor` in muted italic |
| 2 | Devedor | auto | `devedor.nome` ‚Äî bold, color `#0f172a` |
| 3 | Docs | auto | `{num_documentos}` ‚Äî count of documentos; `‚Äî` if 0 |
| 4 | Parcelas | auto | `{num_parcelas_total}x` ‚Äî total parcelas across all docs |
| 5 | Valor Total | auto | `fmtBRL(valor_total)` ‚Äî sum of doc values |
| 6 | Em Atraso | auto | count of parcelas where `saldo_quitado = false` and `data_vencimento < hoje`; `{n} parcelas` or `‚Äî` if zero; shown in `#dc2626` bold when > 0 |

Note: No "Tipo" column at contrato level ‚Äî tipo is per Documento. TabelaContratos omits tipo column entirely.
