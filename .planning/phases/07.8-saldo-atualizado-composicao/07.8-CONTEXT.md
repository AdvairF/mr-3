## 1. Problema

O motor Art.354 em `utils/devedorCalc.js` já calcula correção + multa + juros + honorários. A UI do contrato (`DetalheContrato.jsx:582-598`) só mostra 3 colunas **nominais**: Valor Total, Total Pago, Em Aberto. Advogado não consegue:
- Ver o saldo **atualizado** (com encargos Art.354) no painel do contrato
- Decompor o saldo pra apresentar ao cliente/devedor
- Imprimir a decomposição em formato A4 jurídico

**Consequência**: motor Art.354 NÃO é tocado. Regressão 9/9 preservada por construção (não por teste de mitigação). Phase vira "conectar peças existentes":

```javascript

```
8-12 linhas. Zero mudança no motor original.

**Fix B** — DetalheContrato.jsx:582-598 vira 4ª coluna:
- Grid `repeat(3, 1fr)` → `repeat(4, 1fr)`
- Nova entrada `{ label: "Saldo Atualizado", value: fmtBRL(saldo_atualizado), clickable: true }`
- `saldo_atualizado` via useMemo chamando `calcularDetalheEncargosContrato`
- **Hide a coluna inteira quando `contrato.num_documentos === 0`** (estado válido pós-Phase 7.7)

**Fix C** — Novo `DecomposicaoSaldoModal.jsx`:
- Wrapper em volta de `Modal.jsx` existente (base pronta)
- Body linha-a-linha: Valor Original / + Correção (indexador) / + Multa (%) / + Juros (% a.m.) / + Honorários (%) / [+ Art. 523 se aplicável] / [+ Custas se aplicável] / = Saldo Devedor Atualizado
- **Esconder linhas com valor = 0** (evita poluição)
- **Correção negativa**: sinal `-` + cor vermelha `#dc2626`
- Nota rodapé: "Pode haver discrepância de R$ 0,01 por arredondamento"
- 3 botões: Copiar (clipboard.writeText) / Imprimir (window.open A4) / Fechar

**Fix D** — Impressão isolada via `window.open` (pattern GerarPeticao.jsx):
- `window.open("", "_blank", "width=900,height=700")`
- HTML isolado: `@page size:A4 margin:2.5cm 3cm`, Times New Roman 12pt, line-height 1.8
- Cabeçalho: data + credor + devedor + referência do contrato
- Corpo: decomposição em tabela simples
- Rodapé: linha de assinatura + timestamp
- `.bar` toolbar visível no browser, escondida no print via `@media print { .bar { display: none } }`
